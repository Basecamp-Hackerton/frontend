# Base Camp 경험치 및 보상 시스템

## 📊 개요

Base Camp는 Base 생태계 기여도를 정량화하고 보상하는 종합 커뮤니티 허브입니다. 모든 활동은 경험치(XP)로 환산되며, 이를 통해 Developer Level이 결정됩니다.

---

## 🎯 Developer Level 시스템

### 레벨 구조

Developer Level은 Base 생태계 개발 실력과 기여도를 나타내는 핵심 지표입니다.

- **Level 1-10**: 초보자 (Bronze Tier)
- **Level 11-25**: 중급자 (Silver Tier)
- **Level 26-50**: 고급자 (Gold Tier)
- **Level 51-75**: 전문가 (Platinum Tier)
- **Level 76+**: 마스터 (Diamond Tier)

### 레벨업 공식

```
필요 XP = (현재 레벨 × 500) + (현재 레벨² × 50)
```

예시:
- Level 1 → 2: 550 XP
- Level 10 → 11: 10,000 XP
- Level 25 → 26: 43,750 XP

## 🎯 Base 생태계 기여도 측정

### Developer Level의 의미

Developer Level은 단순한 활동량이 아닌, **Base 생태계에 대한 실제 기여도**를 나타냅니다.

**측정 요소:**
1. **기술적 기여** (40%)
   - 컨트랙트 배포
   - 코드 기여 및 분석
   - 보안 연구
   - 튜토리얼 작성

2. **커뮤니티 기여** (30%)
   - 아티클/게시글 작성
   - 질의응답 기여
   - 멘토링
   - 문서화

3. **네트워크 참여** (15%)
   - DApp 소개 및 후기
   - 이벤트 참여 (AMA, 밋업, 해커톤)
   - Base 캠페인 참여

4. **활동 지속성** (10%)
   - 꾸준한 활동 (출석 체크)
   - 장기 기여
   - 친구 초대 및 커뮤니티 확장

5. **품질** (5%)
   - 받은 추천
   - 인정받은 기여
   - 채택된 답변
