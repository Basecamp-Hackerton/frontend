# Base Camp 커뮤니티 플랫폼 전체 흐름

## 🏗️ 플랫폼 구조

Base Camp는 Base 생태계를 위한 종합 커뮤니티 허브로, 다음 기능들을 통합합니다:

---

## 📱 주요 페이지 및 기능

### 1. 홈 대시보드 (`/`)

**목적**: Base 커뮤니티의 모든 정보를 한눈에 파악

**구성 요소:**
- 공지사항 (최신 3개)
- 최신 게시물 (인기순)
- Base 시세 차트
- 주요 링크 모음
- 빠른 검색

**사용자 액션:**
- 게시판 이동
- 공지사항 확인
- 시세 확인
- 외부 링크 접근

---

### 2. 프로필 페이지 (`/profile`)

**목적**: 개인 활동 기록, 통계, 배지, 레벨 확인

**구성 요소:**
- 프로필 헤더 (아바타, 닉네임, 소셜 링크)
- XP 및 레벨 정보
- 활동 통계
- 활동 캘린더 (깃헙 잔디밭)
- 타임라인 (모든 활동 기록)
- NFT 배지 컬렉션
- 상세 통계

**탭 구성:**
1. **활동 캘린더**: 일별 활동량 시각화
2. **타임라인**: 시간순 활동 기록
3. **NFT 배지**: 획득/미획득 배지
4. **상세 통계**: 카테고리별 상세 통계

---

### 3. 게시판 시스템

#### 3.1 공지사항 (`/board/announcements`)
- 관리자 전용
- 중요 공지 상단 고정
- 카테고리별 분류

#### 3.2 자유게시판 (`/board/free`)
- 일반 게시글
- 댓글, 좋아요
- 추천 시스템

#### 3.3 금융 정보 (`/board/finance`)
- **예측**: Base 관련 가격 예측
- **에어드랍**: 에어드랍 정보 공유
- **자랑글**: 수익 인증, 성공 사례

#### 3.7 DApp 네트워크 (`/board/dapp`)
- **DApp 소개**: Base 관련 DApp 소개
- **사용 후기**: 실제 DApp 사용 후기
- **DApp 발견**: 새로운 DApp 제보

#### 3.4 게임 (`/board/game`)
- **공략**: Base 기반 게임 공략
- **아이템**: 게임 아이템 거래
- **자유**: 게임 관련 자유 토론

#### 3.5 빌드 (`/board/build`)
- 프로젝트 소개
- 개발 일지
- 협업 모집
- 코드 공유

#### 3.6 보안 (`/board/security`)
- **취약점 제보**: 보안 취약점 신고
- **보안 뉴스**: 최신 보안 이슈

---

### 4. 아티클 섹션 (`/articles`)

**목적**: 고품질 기술 문서, 튜토리얼, 백서 요약

**특징:**
- 마크다운 에디터
- 코드 하이라이팅
- 이미지 업로드
- 프리미엄 아티클 (Base 토큰 후원 필요)
- 추천 시스템

**카테고리:**
- 튜토리얼 (LP 기여, 지갑 연결 등)
- 코드 공유 및 분석
- 기술 문서
- 질의응답 모음

**보상:**
- 첫 아티클 작성: NFT
- 추천 50개: +100 XP
- 추천 100개: +200 XP
- 주간 인기 1위: NFT + 500 XP

---

### 5. CTF & 워게임 (`/ctf`)

**목적**: Web3 보안 실력 향상 및 검증

**구성:**
- 문제 목록 (난이도별)
- 문제 상세 페이지
- 제출 및 검증
- 리더보드
- 해결 인증 (개인 인증 필요)

**보상:**
- 문제 해결: 150 XP
- CTF 5문제: 배지
- CTF 20문제: 레어 배지

---

### 6. 개발 도구 (`/tools`)

**목적**: Base 개발에 필요한 도구 제공

**도구 목록:**
- 컨트랙트 배포 도구
- 가스 계산기
- ABI 인코더/디코더
- 트랜잭션 검증기
- 테스트넷 파우셋

---

### 7. 리더보드 (`/leaderboard`)

**목적**: 커뮤니티 활동 순위 확인

**리더보드 종류:**
- 전체 리더보드 (Total XP)
- 주간 리더보드
- 월간 리더보드
- Developer Level 리더보드
- 카테고리별 리더보드

---

### 8. 공식 자료 허브 (`/resources`)

**목적**: Base 관련 공식/비공식 자료 통합

**자료 종류:**
- Base 공식 문서
- 백서 및 기술 문서
- 테스트넷 빌드 가이드
- 보안 뉴스
- 공식 포럼 링크
- 밋업/이벤트 정보
- 외부 서비스 링크

### 9. 이벤트 섹션 (`/events`)

**목적**: Base 관련 이벤트 정보 및 참여

**이벤트 종류:**
- **AMA**: Base 팀 AMA 일정 및 참여
- **밋업**: 온라인/오프라인 밋업 정보
- **Base 캠페인**: 공식 캠페인 참여 인증
- **해커톤**: Base 해커톤 정보 및 프로젝트 제출
- **커뮤니티 이벤트**: Base Camp 자체 이벤트

**참여 보상:**
- AMA 참여: 100 XP (질문 등록 +50 XP)
- 밋업 참여: 150 XP (발표 시 +300 XP)
- 캠페인 참여: 200 XP
- 해커톤 참여: 500 XP (수상 시 +1000 XP)

### 10. 친구 초대 시스템 (`/referral`)

**목적**: 커뮤니티 확장 및 네트워크 효과

**기능:**
- 고유 초대 링크 생성
- 초대 현황 대시보드
- 초대 보상 추적
- 친구 활동 모니터링

**보상:**
- 친구 초대: 50 XP
- 초대받은 친구 첫 활동: +100 XP
- 초대받은 친구 레벨업: +50 XP
- 5명 초대: +200 XP
- 10명 초대: +500 XP + NFT 배지
- 50명 초대: +2000 XP + 레어 NFT 배지

## 🔄 사용자 여정 (User Journey)

### 신규 사용자

1. **지갑 연결** → EOA 지갑으로 로그인 (+5 XP, 첫 로그인 배지)
2. **프로필 설정** → 닉네임, 아바타, 소셜 링크 입력
3. **출석 체크** → 일일 출석 체크 (+5 XP)
4. **첫 활동** → 게시글 작성 (+100 XP, "First Post" 배지)
5. **네트워크 참여** → DApp 소개 또는 후기 작성
6. **지식 기여** → 튜토리얼 작성 또는 코드 공유
7. **이벤트 참여** → AMA, 밋업, 해커톤 참여
8. **친구 초대** → 초대 링크 공유
9. **레벨업** → XP 획득으로 레벨 상승

### 활성 사용자

1. **일일 활동** → 출석 체크, 게시글, 댓글, 좋아요
2. **네트워크 참여** → DApp 후기, 이벤트 참여
3. **지식 기여** → 튜토리얼, 코드 공유, 질의응답
4. **주간 목표** → 주간 리더보드 진입
5. **개발 활동** → 컨트랙트 배포, CTF 해결
6. **커뮤니티 기여** → 아티클 작성, 멘토링
7. **친구 초대** → 커뮤니티 확장
8. **레벨 상승** → Developer Level 향상

### 파워 유저

1. **월간 리더보드** → 상위권 유지
2. **특별 배지** → 레전더리 배지 획득
3. **커뮤니티 리더** → 멘토, 관리자 역할
4. **Base 협업** → Base Foundation과 협업 기회
